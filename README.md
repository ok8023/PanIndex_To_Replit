# PanIndex-Replit
在Replit构建PanIndex网盘工具

Replit创建Bash语言，shell中输入

```
bash <(curl -s https://raw.githubusercontent.com/ok8023/PanIndex_To_Replit/main/install.sh)
```

回车等待输出完毕

输入

```
sh install.sh
```

回车等待输出完毕

点击绿色Run按钮运行
